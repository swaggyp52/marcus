## Interaction Patterns

### Hover States
- Buttons: Darker shade, slight lift (box-shadow)
- Cards: Subtle background change, border highlight
- Links: Underline appears

### Click States
- Buttons: Pressed appearance (darker, no shadow)
- Links: Color change

### Loading States
- Content: Spinner animation
- Form: Disabled button, "Sending..." text
- API Calls: Network badge shows pending

### Error States
- Invalid input: Red border, error message below
- API error: Red alert box with error text
- 404 error: "Resource not found" message

### Success States
- Form submit: Green success alert, "âœ“ Saved"
- Upload: Green progress bar to 100%
- Item created: Refresh list, highlight new item
